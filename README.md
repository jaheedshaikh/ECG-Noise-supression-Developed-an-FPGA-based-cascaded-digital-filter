# ECG-Noise-supression-Developed-an-FPGA-based-cascaded-digital-filter
Developed an FPGA-based cascaded digital filtering system for ECG signal denoising by implementing high-pass, notch, FIR low-pass, and Savitzky–Golay filters to effectively remove baseline wander, power line interference, and high-frequency noise, thereby improving overall signal quality and accuracy.
